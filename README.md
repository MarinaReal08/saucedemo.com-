# Тесты логина для https://www.saucedemo.com/

Автоматизированные тесты авторизации на сайте **SauceDemo** с использованием **Python + Playwright + Pytest + Allure + Page Object**.  
Проект включает 5 тестов, проверяющих разные сценарии логина.

---

## Проверяемые сценарии

- Успешный логин (`standard_user / secret_sauce`)
- Логин с неверным паролем
- Логин заблокированного пользователя (`locked_out_user`)
- Логин с пустыми полями
- Логин пользователя `performance_glitch_user` (страница должна открыться несмотря на задержку)

---

## Требования

- Python 3.10  
- Установленный **Allure Commandline** (для просмотра отчётов)
